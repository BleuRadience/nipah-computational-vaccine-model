## Overview

Comprehensive pipeline for designing multi-epitope subunit vaccine candidates targeting Nipah virus attachment glycoprotein G. Integrates authentic NCBI sequences, hybrid epitope prediction, structural simulation, and immunoinformatics analysis.

## Quick Start

```bash
# Install dependencies
pip install -r requirements.txt

# Run analysis
python nipah_vaccine_model.py

# Quick mode
python nipah_vaccine_model.py --quick --output results
```

## Outputs
- `comprehensive_report.txt` - Detailed analysis summary
- `vaccine_construct.fasta` - Multi-epitope sequence
- `comprehensive_analysis.png` - Key visualizations
- `cloning_sequence.txt` - E. coli optimized DNA

## Scientific Context

Addresses critical need for Nipah virus countermeasures:
- 40-75% case fatality ratio
- No approved vaccines (as of Feb 2026)
- Recent outbreaks (West Bengal 2025-2026)
- Complements ongoing Oxford ChAdOx1 NipahB Phase II trial
